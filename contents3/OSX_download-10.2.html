Installing VPython on Mac OSX 10.2 running X11

Important: This is a limited, out of date version and is no longer supported.
Go to vpython.org

The installation instructions below assume that you have administrative permissions to install new software; you will be asked for your password at various stages of the installation. If you prefer, you may find it useful to be the "root" user when installing the software, but because of the dangers of operating as root, Apple hid the root account in OS X. To enable it, boot off the install CD (by holding down the "C" key), and look under the Install menu for the "Password Reset" option. You can use this to change your own password if you forget it, and to enable the root account.

1) Go to http://www.apple.com/macosx/features/x11/download and download both X11 and SDK (includes necessary libraries)

2) Install fink (a program used to install Unix-based OSX applications):

Follow the instructions you find at http://fink.sourceforge.net. Answer "y" when asked about adding a line to ~/.cshrc. You do not need to run the "dselect" program that fink installs.

3) Install Python 2.2:
In a "terminal emulator" or "typescript", type the following commands.
rehash (make sure the system knows about the recent installations)
In the following, answer "y" when asked about installing dependency packages.
   sudo apt-get update (updates Fink's catalog of packages)
   sudo apt-get install pkgconfig (configuration package needed by VPython installer)
   sudo apt-get install gtk+ (graphics toolkit needed by VPython)
   sudo apt-get install gtkglarea (more graphics needed by VPython)
   sudo apt-get install python22 (Python 2.2 for Mac OSX)
rehash (make sure the system knows about the recent installations)
which python2.2 (you should see /sw/bin/python2.2)
   If not, your search path is not giving preference to /sw/bin, and installation won't work properly.
python2.2 (you should see "Python 2.2"; type 2+2, press Enter, and see 4)
Ctrl-D to exit from Python

4) Install Visual and other needed components:

Download VPython installer VPythonOSX-20031017.tar.gz
The download process normally puts on the desktop the *gz file, a *.tar file, and an unpacked folder.
Discard the *.gz file and the unpacked folder.
Move the *.tar file into your own folder.
Open a typescript and cd to your folder.
sudo tar -Pxf VPythonOSX-20031017.tar (extract files into /sw/lib and /sw/bin)
   (If you start from the .gz file, you say -Pxzf instead of -Pxf, in order to uncompress the .gz file.)
The following components were installed into /sw/lib/python2.2/site-packages/:
visual (the *.py components of Visual) and cvisualmodule.so (the main Visual engine) 
3D demo programs
IDLE for VPython (convenient edit/run environment for writing and testing programs)
documentation for Visual
Numeric module which Visual uses
A convenience shell script vpython was installed in /sw/bin/vpython.
 
Edit your environment file ~/.cshrc to include the following:
        setenv BROWSER open (to use Safari to read the on-line help)
You can check this is set with "echo $BROWSER".
 
5) Running VPython:
To run VPython, start up X11, and in a typescript execute this:
vpython (starts the IDLE for VPython program editor, starting in the VPython demos directory)
Open demo file bounce2.py in demos directory, press F5.
   You should see a ball bouncing around in a box.
   Right button rotates the camera; middle (left+right) button zooms.

Updating one or more components of VPython:

After installing Python 2.2 and VPython, the following individual component zip files can be downloaded and unzipped. These components are included with the complete VPython package but can be updated without reinstalling the whole package.

Demos, Documentation, Visual module, IDLE for any platform
